# TS Config Plugin

Just install the VSIX [packge](https://github.com/octref/ts-config-plugin/releases/tag/v0.1.6).

Or, `yarn add ts-config-plugin` and put this in tsconfig:

```json
{
  "compilerOptions": {
    "plugins": [
      { "name": "ts-config-plugin" }
    ]
  }
}
```
